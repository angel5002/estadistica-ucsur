# Proyecto: Pagina_Estadistica

## Design System
Antes de escribir cualquier UI, leer y aplicar las pautas de **DESIGN.md**.
El diseño está inspirado en claude.ai: paleta cálida (parchment `#f5f4ed`), tipografía serif, acento terracota `#c96442`, neutros warm-toned.

Combinar siempre con las skills **frontend-design** y **frontend-patterns** para estructurar componentes, layouts y patrones de interacción de manera consistente.
